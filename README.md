# AI社交自动化助手

> 小红书自动化操作助手，支持自动浏览、点赞、评论等功能

## 📋 项目简介

AI社交自动化助手是一款基于Chromium浏览器扩展开发的小红书自动化工具，旨在帮助用户更高效地浏览和互动小红书内容。

### ✨ 功能特性

| 功能 | 说明 | 默认状态 |
|------|------|---------|
| 自动选取笔记 | 根据关键词自动筛选并点击目标笔记 | ✅ 开启 |
| 自动点赞 | 进入笔记后自动点赞 | ✅ 开启 |
| AI智能评论 | 使用豆包AI生成自然评论内容 | ❌ 关闭 |
| 自动下滑探索 | 自动滚动加载更多内容 | ✅ 开启 |
| 关键词筛选 | 支持自定义目标关键词和排除词 | ✅ 开启 |

## 🛠️ 技术栈

- **开发语言**: JavaScript (ES6+)
- **运行环境**: Google Chrome / Microsoft Edge 浏览器（基于Chromium内核）
- **API服务**: 火山方舟豆包API
  - **API地址**: `https://ark.cn-beijing.volces.com/api/v3/chat/completions`
  - **模型名称**: `doubao-seed-2-0-pro-260215`
  - **端点ID**: `ep-m-20260420182011-v5xsw`

## 📦 安装部署

### 1. 下载扩展

将扩展文件夹 `ai-social-helper` 下载到本地计算机。

### 2. 安装到浏览器

#### Google Chrome 浏览器
1. 打开 Google Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 开启右上角的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择下载的 `ai-social-helper` 文件夹
6. 安装成功后，浏览器右上角会出现扩展图标

#### Microsoft Edge 浏览器
1. 打开 Microsoft Edge 浏览器
2. 在地址栏输入 `edge://extensions/` 并回车
3. 开启左侧的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择下载的 `ai-social-helper` 文件夹
6. 安装成功后，浏览器右上角会出现扩展图标

### 3. 配置API Key（可选）

如果需要使用AI智能评论功能，请配置火山方舟API Key：

1. 点击浏览器右上角的扩展图标
2. 在弹出的配置面板中填入您的API Key
3. 勾选 **AI智能评论** 选项
4. 点击 **保存配置**

**获取API Key**:
- 访问 [火山方舟开放平台](https://www.doubao.com/)
- 注册账号并创建应用
- 获取API Key（格式：`ark-xxxxxxxxxxxx`）

### 4. 更换AI模型（可选）

本扩展默认使用 `doubao-seed-2-0-pro-260215` 模型，您可以根据需要更换为其他火山方舟平台支持的模型：

**支持的其他模型**:
- `doubao-3-0-pro`
- `doubao-3-0-mini`
- `doubao-3-0-turbo`
- 其他火山方舟平台支持的模型

**更换方法**:
在 `content.js` 文件中找到 `generateAIComment` 函数，修改以下参数：
```javascript
body: JSON.stringify({
  model: 'doubao-seed-2-0-pro-260215', // 替换为其他模型名称
  endpoint_id: 'ep-m-20260420182011-v5xsw', // 替换为对应端点
  // ...
})
```

## 🚀 使用说明

### 基本操作流程

1. **访问小红书**
   - 打开浏览器，访问 [小红书首页](https://www.xiaohongshu.com/explore) 或 [搜索页面](https://www.xiaohongshu.com/search_result?keyword=xxx)

2. **启动扩展**
   - 扩展会自动检测页面类型
   - 在首页/搜索页自动启动滚动和筛选功能

3. **自动化流程**
   ```
   下滑×2 → 筛选关键词 → 点击目标笔记 → 点赞 → 评论 → 返回 → 重复
   ```

### 配置选项

点击扩展图标打开配置面板：

| 选项 | 说明 |
|------|------|
| **火山方舟API Key** | AI评论必需的API密钥 |
| **自动点赞** | 进入笔记后自动点赞 |
| **AI智能评论** | 使用AI生成评论内容 |
| **自动选取笔记** | 自动识别并点击意向笔记 |
| **自动下滑探索** | 自动滚动加载更多内容 |

### 关键词配置

扩展内置了丰富的关键词库，支持以下类型：

**目标关键词**（优先匹配）:
- 身材相关：腿、美腿、长腿、丝袜、黑丝...
- 人物描述：美女、辣妹、女神、小姐姐...
- 穿搭相关：穿搭、ootd、显瘦穿搭...
- 身高参数：170-190身高数字...
- 身份相关：女友、女朋友、情侣...
- 肤色相关：显白、冷白、冷白皮...

**排除关键词**（自动过滤）:
- 医美类：医美、整形、玻尿酸、热玛吉...
- 做饭类：做饭、烹饪、食谱、烘焙...
- 其他：职场、考研、数码、汽车...

## ⚙️ 核心配置说明

### manifest.json

```json
{
  "manifest_version": 3,
  "name": "AI社交自动化助手",
  "version": "1.0.0",
  "description": "小红书自动化操作助手",
  "permissions": ["storage"],
  "content_scripts": [
    {
      "matches": ["*://*.xiaohongshu.com/*"],
      "js": ["content.js"]
    }
  ],
  "action": {
    "default_popup": "popup.html"
  },
  "background": {
    "service_worker": "background.js"
  }
}
```

### content.js 核心功能

| 函数 | 功能 |
|------|------|
| `isHomePage()` | 判断是否为主页/搜索页 |
| `isNoteDetailPage()` | 判断是否为笔记详情页 |
| `isTargetNote()` | 判断笔记是否匹配目标关键词 |
| `findAndClickTargetNote()` | 查找并点击目标笔记 |
| `doLike()` | 自动点赞 |
| `doComment()` | 自动评论 |
| `closeNote()` | 返回主页 |
